Notes

Object Methods : this

  • Function that are stored in object properties are called "methods".
  • Methods allow objects to "act" like object.doSomething().
  • The value of this is defined at run-time.
  • When a function is declared, it may use this, but that this has no value until the function is called.
  • When a function is called in the "method" syntax: object.method(), the value of this during the call is object.

Chapter 5: Objects

  • One way to think about an object is that it's like a dictionary where you look up a property name and see a value.
  • Objects are often used to keep any related information and functionality togather in the same place.
  • To create an object literal, simply enter a pair of curly braces {}. Example: const spiderman = {};
  • You can access the properties of an object using the dot notation. Example: superman.name
  • The beauty of JSON is that it manages to hit the sweet spot between being both human and machine-readable.
  • A constructor function is used to create a new date object using the new operator. Example: const today = new Date();

Chapter 6: Document Object Model

  • The Document Object Model, or DOM for short, represents an HTML document as a network of connected nodes that form a tree-like structure.
  • The DOM provides several methods that allow us to access any element on a page.
  • The getElementById() method returns a reference to the element with a unique id attribute that is given as an argument.
  • getElementByTagName() will return a live node list of all the elements with the tag name that is providad as an argument.
  • getElementByClassName() will return a live node list of all elements that have the class name that is supplied as an argument.
  • The document.querySelector() method allows you to use CSS notation to find the first element in the document that matches a CSS selector provided as an argument.

Chapter 7: Events

  • Events are like setting a notification to alert you when something happens.
  • Events listeners are added to elements on the page and are part of the DOM.
  • The addEventListener() method is called on a node object, the node to which the event listener is being applied. Example: document.body.addEventListener('click',doSomething);
