Chapter 8: Transforms and Transitions

  • Transform: The CSS3 transform property allow you to translate, rotate, scale, and/or skew any element on the page.
  • You can move elements left, right, up, or down. Similary to the behavior of position: relative; when declaring top and left, moving elements up and down or left and right along the x and y axes.
  • The scale(x,y) function scales an element by the defined factors horizontally then vertically. If only one value is provided, it will be used for both the x and y values
  • The rotate() function rotates an element around the point of origin by a specified angle value.
  • The skew(x,y) function specifies a skew along the x and y axes. As you'd expect, the x specifies the skew on the x axis, and the y specifies the skew on the y axis.
  • The transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state. You can specify this either in seconds ( s ) or milliseconds ( ms ).
  • The transition-timing-function lets you control the pace of the transition in even more granular detail.
  • By using the transition-delay property, it's possible to introduce a delay before the transition begins. Normally a transition begins immediately, so the default is 0 . Include the number of milliseconds ( ms ) or seconds ( s ) to delay the transition.
  • Animation-name: This property is used to attach an animation (previously defined using the @keyframes syntax) to an element.
  • Animation-duration: The animation-duration property defines the length of time (in seconds or milliseconds) an animation takes to complete one iteration (all the way through, from 0% to 100%)
  • Animation-timing-function: the animation-timing-function determines how the animation will progress over its duration.

Chapter 12: Canvas, SVG, and Drag and Drop

  • With HTML5's Canvas API, we can draw anything we can imagine, all through JavaScript.
  • The text in between the canvas tags will only be shown if the canvas element is not supported by the visitor's browser.
  • Once a canvas element is stored in a js variable, we MUST setup the context. The context is the place where your drawing is rendered.
  • Using fillReact and strokeReact methods, we are able to draw a rectangle on the canvas.
  • To draw a circle, we use the beginPath(), arc(), and closePath methods.
  • To save a Canvas drawing, meaning to store it locally, we can use the toDataUrl method. This method creates a URL with the image in it.
  • SVG stands for a scalable vector graphic. Instead of pixels, each "pixel" works as a vector using vector math. It scales and doesn't go blurry.
