Programming Basics

  1. Variables
    • Variables have to be declared before they can be used.
    • The keyword const is used when the variable will not be reassigned to another value.
    • Let is used if the variable might be reassigned later in the program.
    • Var worked in much the same way as using let. The main difference was that variables declared using let and const have block scope.
  2. Scope
    • Global scope means it is accessible everywhere in the program.
    • Local scope means that any variable defined inside a block using the let and const will only be available inside that block and not be accessible outside pf that block.
