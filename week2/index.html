Programming Basics

  1. Variables
    • Variables have to be declared before they can be used.
    • The keyword const is used when the variable will not be reassigned to another value.
    • Let is used if the variable might be reassigned later in the program.
    • Var worked in much the same way as using let. The main difference was that variables declared using let and const have block scope.
  2. Scope
    • Global scope means it is accessible everywhere in the program.
    • Local scope means that any variable defined inside a block using the let and const will only be available inside that block and not be accessible outside pf that block.
  3. Template Literal
    • Template Literals are a special type of string that were introduced in ESG.
    • Template Literals use the backtick character ` to deliminate the string.
    • Template Literals allow interpolation of JavaScript code. This means that a JavaScript expression can be inserted inside a string and the result will be displayed, as can be seen in this example:

      • const name=`Siri`;
      `Hello${name}¡`;
      >>'Hello Siri¡'

    • This JS expression is placed inside the curly braces with a$ character in fron of them.

Arrays, Logic, and loops

  • An array is an ordered list of values. To create an array literal, simple write a pair of queare brackets: const myArray=[];
  • The pop()method removes the last item from an array.
  • The shift()method removes the first item in the array.
  • The push()method appends a new value to the end of the array.
  • The unshift()method appends a new item to the beginning of the array.
  • The join()method can be used to turn the array into a string that comprises all the items in the array separated by commas.

Functions

  1. To define a function literal we can use the function declaration.
  2. Another way of defining a function literal is to create a function expression.
  3. A function can also be declared using the contructor Function().The body of the function is entered as a string
