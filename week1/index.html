Notes

This week I spent time reading on the book "Doing Stuff with Web Thing" by Lee S. Barney. I review some HTML, CSS, and JavaScript concepts that I took in previous courses.

Using JavaScript you can store any information you want to use later. Even if your user refreshes the page or goes to some other site and then comes back. We called LocalStorage to code that stores the story.

JavaScript's local storage uses what techies call key-value pairs. An old fashioned phone book is the best example to understand this principle. The key would be the person's name and the value would be their address and phone number.
